# Modelos y Simulacion

Bases para los 6 proyectos.

## Algoritmos implementados

### 1) Congruencial multiplicativo

Archivo:

`congruencial_multiplicativo/congruencial_multiplicativo.py`

Ejecucion:

`python congruencial_multiplicativo/congruencial_multiplicativo.py 7 5 32 10`

Modo interactivo (teclado):

`python congruencial_multiplicativo/congruencial_multiplicativo.py`

Parametros:

- `semilla`: valor inicial `X0`
- `a`: multiplicador
- `m`: modulo
- `n`: cantidad de numeros a generar

### 2) Producto medio

Archivo:

`producto_medio/producto_medio.py`

Ejecucion:

`python producto_medio/producto_medio.py 5015 5734 4 10`

Modo interactivo (teclado):

`python producto_medio/producto_medio.py`

Parametros:

- `x0`: primera semilla
- `x1`: segunda semilla
- `d`: cantidad de digitos por semilla
- `n`: cantidad de numeros a generar
